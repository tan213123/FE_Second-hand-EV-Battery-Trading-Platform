*Nền tảng thương mại điện tử chuyên biệt cho Xe điện cũ và Pin đã qua sử dụng.*

##  Giới thiệu 
Dự án được xây dựng nhằm giải quyết bài toán kết nối mua bán **xe điện và pin** đã qua sử dụng chất lượng cao. Đây là sản phẩm demo thể hiện kỹ năng lập trình **Frontend**, tư duy thiết kế giao diện (UI/UX) và tổ chức source code hiện đại.

##  Tính năng chính 
* ** Đăng ký/Đăng nhập:** Phân quyền chặt chẽ giữa Người dùng (User) và Quản trị viên (Admin).
* ** Quản lý tin đăng (CRUD):** Cho phép người dùng đăng bán xe/pin, hỗ trợ tải lên (Upload) và xem trước (Preview) hình ảnh.
* ** Admin Dashboard:** Trang quản trị chuyên biệt để kiểm soát nội dung và thống kê hệ thống.
* ** Tiện ích người dùng:** Chức năng tìm kiếm, xem chi tiết, lưu tin yêu thích và **So sánh** thông số kỹ thuật.

##  Tech Stack 

### Frontend & Core
* **Core:** ReactJS (Build tool: **Vite** ).
* **Language:** JavaScript (**ES6+**).
* **Styling:** CSS3 / SCSS (Module).
* **State Management:** Context API.

### Backend & Integration
* **BaaS:** **Supabase** (Database & Auth).
* **API Client:** Axios.
* **Testing:** REST Client (Test API trực tiếp trên VSCode).

### Tools & Quality Control
* **Version Control:** Git / Github.
* **Code Quality:** **ESLint**.
* **Design:** Figma.
